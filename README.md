## Projects

### [Stripe Clone — Payment Processing System](https://github.com/sampom7)
`Java` `Spring Boot` `PostgreSQL` `Redis` `Apache Kafka` `Docker`

Production-grade payment backend modelled after Stripe's core infrastructure.
- Complete payment lifecycle: Initiation → Authorization → Capture → Settlement
- Double-entry ledger accounting for financial correctness and audit integrity
- Idempotent REST APIs with Redis caching to safely handle retries
- Kafka-based event processing with exactly-once semantics
- Optimistic + pessimistic locking, fully Dockerized infra (PostgreSQL, Redis, Kafka, Zookeeper)

---

### [AI Intrusion Detection System for WSNs](https://github.com/sampom7)
`Python` `Flask` `Scikit-learn` `Random Forest` `Chart.js`

End-to-end security system for Wireless Sensor Networks — final year B.Tech project.
- **99.7% detection accuracy** across Blackhole, Grayhole, Flooding, and TDMA attack classes
- Real-time backend simulation engine with live ML inference exposed via Flask REST APIs
- Autonomous mitigation engine triggering node isolation and traffic control by threat severity
- SOC-style dashboard with live network topology, threat alerts, and forensics logging

### [Netflix Ratings — Chrome Extension](https://github.com/sampom7/netflix-ratings-extension/)
`JavaScript` `ManifestV3` `REST APIs` `DOM Manipulation`
